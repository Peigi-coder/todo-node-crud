# ToDo CRUD – Node.js + Express + SQLite

Mini-aplikacja CRUD w **Node.js + Express + SQLite**  
Frontend – **HTML + JavaScript (SPA)**.  
Encja domenowa: **Task** z polami:
- `id` (int, PK)
- `title` (text)
- `description` (text)
- `priority` (int 1–5)
- `deadline` (date, YYYY-MM-DD)
- `done` (bool)
- `created_at` (datetime)

## 1. Uruchomienie lokalne

```bash
npm install
npm start
